# scott-financas
Sistema web de gestão financeira desenvolvido em Java (Servlets), HTML e SQL como projeto acadêmico de Análise e Desenvolvimento de Sistemas.

## Tecnologias utilizadas

- Java
- Servlets
- HTML
- SQL
- Maven

## Funcionalidades

- Cadastro de usuários
- Sistema de login
- Controle de movimentações financeiras
- Visualização de dados em dashboard

## Objetivo

Projeto acadêmico com objetivo de aplicar conceitos de desenvolvimento web, backend em Java e integração com banco de dados.
